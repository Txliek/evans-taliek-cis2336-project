# ArtConnect

## Description
ArtConnect is a front-end web application created for the CIS 2336 course. The website allows users to browse artwork, view upcoming art events, submit artwork information through a form, and access frequently asked questions. The project focuses on creating a responsive, user-friendly interface using HTML, CSS, and JavaScript.

## Features
- Homepage with featured artwork and events
- Gallery page with artwork listings
- Events page displaying upcoming art events
- Artist submission form with client-side validation
- FAQ page with expandable and collapsible questions
- References page documenting sources and AI prompts
- Responsive design for different screen sizes

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git
- GitHub Pages
